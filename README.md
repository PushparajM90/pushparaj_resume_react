## 📦 Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/portfolio.git
cd portfolio
```

### 2. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### 3. Backend Setup (if using Django)

```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

---

## ▶️ Usage

1. Run frontend server → http://localhost:5173
2. Run backend server → http://127.0.0.1:8000
3. Open the portfolio in your browser
4. Explore projects, skills, and contact details
